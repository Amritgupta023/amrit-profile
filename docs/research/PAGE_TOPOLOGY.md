# Page Topology

Target: `https://brittanychiang.com/`

- Desktop layout: two-column shell, max width about 1280px, 96px horizontal padding.
- Left column: sticky full-height intro with name, role, tagline, section nav, and social links.
- Right column: scrollable content with About, Experience, Projects, Writing, and Footer.
- Mobile/tablet layout: intro becomes normal flow, section nav hidden, headings become sticky section labels.
- Interaction model: scroll-driven active section nav, smooth anchor scrolling, pointer-driven radial spotlight, hover-highlighted row cards and links.
- Assets: no content images required for the main clone; original is mostly text, links, pills, and SVG/social link affordances.
